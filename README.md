# 💰 SmartRetireAI

An AI-powered retirement planning assistant built using Python, Machine Learning, and Streamlit.

## 📌 Overview
SmartRetireAI helps users estimate whether they can retire safely by analyzing:

- Monthly Salary
- Monthly Expenses
- Current Savings
- Dependents
- Risk Tolerance
- Expected Return
- Inflation Rate
- Desired Retirement Age

The system predicts:

✅ Required Retirement Corpus  
✅ Monthly Saving Target  
✅ Retirement Safety Score  
✅ Risk Analysis (Low / Medium / High)  
✅ Income Split Visualization  
✅ AI-based Financial Recommendations  
✅ Downloadable Retirement PDF Report

## 🧠 Machine Learning
We generated 10,000 synthetic financial profiles and trained a machine learning model to predict retirement corpus requirements based on user financial behavior.

Model used:
- Random Forest Regressor
